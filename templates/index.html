Automatic Bug Reporting Tool

ABRT is a set of tools for detecting and reporting application crashes.
Crash can be automatically reported without human intervation.
Powerusers can add addional informations.
Reports are aggregated based on similarities for developers.

Fedora Analysis Framework

Web pages containing graphs and reports of crashes.
You can easily see when and how often application crashes.
Developers immediately see most frequent crashes.

Retrace Server
